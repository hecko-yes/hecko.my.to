Unofficial Mitsua utility page

ミツアの非公式ユーティリティページ

These utilities were made by hecko for contributors to the Elan Mitsua project, which creates image generation models trained without using unauthorized copyrighted data. This page is not affiliated with the project and all opinions expressed are solely of the author.

これらのユーティリティは、絵藍ミツアプロジェクトの貢献者のためにヘッコちゃんによって作成され、無許可の著作権データを使用していない画像生成モデルを作成する。このページはプロジェクトに付属しておらず、意見はすべて作者のものです。

This page is marked with the CC0 1.0 Universal public-domain dedication. No generative large language models were used in the creation of this page. Since this page is dedicated to a model trained only on authorized data, the Japanese translation was done using ElanMT. (Manual dictionary lookup and backtranslation was done for verification.) I hope this proves that you don't need to scrape the entire internet to train a useful model.

このページには、CC0 1.0全世界パブリックドメインの献辞が表示されています。このページは、許可されたデータのみに学習されたモデルに特化しているため、日本語訳は、ElanMTを使用して行われた。（検証のために手動辞書の検索と逆翻訳が行われた。）これは、便利なモデルを訓練するにはインターネット全体を削る必要がないことを証明してほしいのですが。

Advice

助言

  • If Mitsua learns too many similar variants of one image, she might memorize the image and recreate it almost exactly. Transparent backgrounds and superlikes can intensify this, as they cause Mitsua to study the image more times. For example, the prompt elanmitsua, fairy will often cause Mitsua to replicate one particular image of Mitsua and Zundamon, which was submitted with 10 variants with transparent backgrounds, each of which received ~4 superlikes. ミツアが1つの画像の類似の変異体を多すぎることを学んだ場合、彼女はその画像を暗記し、ほぼ正確に再現する可能性がある。透明な背景やスーパーライクは、ミツアが画像をよりよく研究するようにするため、この傾向を強めることができる。たとえば、プロンプトelanmitsua, fairyは、多くの場合、ミツアがミツアとズンダモンの1つの特定のイメージを複製することになり、透明の背景を備えた10の変種が提出され、それぞれが約4つのスーパーライクを受け取った。これを良いことと悪いこととは、この情報は賢明に使いなさい。
